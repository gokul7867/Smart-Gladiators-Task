# 📱 Quotes Camera App

## Features

### Task 1
- Fetch quotes from API
- Display using UICollectionView
- Dynamic cell height

### Task 2
- Custom camera using AVCaptureSession
- Capture photo
- Preview in next screen

## Tech Stack
- Swift
- UIKit (XIB)
- AVFoundation

## Notes
- Clean architecture followed
- No SwiftUI used
